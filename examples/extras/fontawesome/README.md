# Extra - Fontawesome

To install FontAwesome, add the following code to your `index.html`

```html
<!DOCTYPE html>
<html lang="en">
    <head>
        <!-- ... -->
        
        <!-- Font Awesome -->
        <link rel="preload" href="/extras/fontawesome/css/fontawesome.min.css" as="style" onload="this.onload=null; this.rel='stylesheet';">
    </head>
    <body>
        <!-- ... -->
    </body>
</html>
```
